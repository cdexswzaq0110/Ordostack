# OrdoStack Logo Direction

## Selected direction

The selected logo direction is concept A: a balanced hanging mobile.

It maps the product idea to a simple visual system:

- suspended arcs = daily scheduling flow
- leaf shapes = tasks and priorities
- center vertical line = the current plan anchor
- asymmetry = realistic day planning, not a rigid grid

## Palette

| Token | Hex | Usage |
| --- | --- | --- |
| Ink | `#111111` | wordmark, dark icon background |

| Muted taupe | `#6f6964` | light logo linework |

| Terracotta | `#b93a2f` | priority accent leaf |
| Frame grey | `#a39e9a` | secondary task leaf |

## Usage

- Use `ordostack-mark.svg` for app icons and small placements on light backgrounds.
- Use `ordostack-icon-dark.svg` inside the dark sidebar or on dark surfaces.
- Use `ordostack-logo-horizontal.svg` for README, landing pages, and brand headers.
